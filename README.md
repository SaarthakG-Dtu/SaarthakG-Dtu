## Selected Projects

### Semantic Segmentation on Aerial Drone Imagery

Developed a high-resolution semantic segmentation pipeline tailored for aerial datasets with fine-grained object distributions.

* Implemented a UNet-based architecture with a transfer learning backbone in PyTorch
* Designed a custom patching and reconstruction workflow for large-scale imagery
* Addressed severe class imbalance using Dice + Cross-Entropy loss
* Optimized training using AdamW and OneCycleLR scheduling
  Focused on improving small-object detection in complex aerial scenes.

---

### Vision-Based Fruit Counting using YOLOv8 and OpenCV

Designed a dual-view fruit detection and counting system combining geometric alignment and vision-based detection.

* Applied ORB feature matching and RANSAC for homography alignment
* Used HSV color masking for robust multi-color fruit detection
* Built custom cross-view matching logic using spatial proximity
* Integrated YOLOv8 for reliable multi-object detection
  Emphasis was on practical computer vision system design beyond single-frame inference.

---

### Darpa Triage Challenge

Explored perception pipelines combining motion, depth, and visual signals for real-time understanding.

* Dense pixel tracking for motion feature extraction using LiDAR and depth camera.
* Feature engineering using displacement and angular flow patterns
* LSTM-based temporal modelling for sequential motor signal analysis
* Investigation of sensor fusion strategies for perception systems
